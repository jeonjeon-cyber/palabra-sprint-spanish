# Palabra Sprint

스페인어 단어 학습용 정적 웹앱입니다.

## 공유 링크로 배포하기

이 프로젝트는 `GitHub Pages`에 바로 배포할 수 있게 설정되어 있습니다.

### 1. GitHub 새 저장소 만들기

- GitHub에서 새 저장소를 하나 만듭니다.
- 저장소 이름은 예를 들어 `palabra-sprint`로 해도 됩니다.

### 2. 이 폴더 파일 올리기

이 폴더 안 파일들을 만든 저장소에 업로드합니다.

필수 파일:

- `index.html`
- `styles.css`
- `app.js`
- `.github/workflows/deploy-pages.yml`
- `.nojekyll`

### 3. GitHub Pages 켜기

저장소에서:

1. `Settings`
2. `Pages`
3. `Source`는 `GitHub Actions` 선택

### 4. 자동 배포 완료 기다리기

코드를 `main` 또는 `master` 브랜치에 올리면 자동으로 배포됩니다.

배포가 끝나면 링크가 생깁니다.

형식 예시:

`https://<github-username>.github.io/<repository-name>/`

## 로컬 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.
